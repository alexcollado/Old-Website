Projects

    • Metro Map Maker
      Java Project

      For my final project in my third-level Obect-Oriented Programming and Design Patterns class, I created a Metro Map Maker that allows users to create their own metro maps using stations and lines. The project had to utilize JavaFX, and my professor provided skeleton frameworks that were used as a jump-start for the application. I implemented local saving and loading using JSON.

      Before creating the application, I wrote a mock Software Design Document (SDD) as if I were a developer from another company. The SDD detailed the package, class, and method-level design viewpoints, as well as the file structure and formats of the application.

      In addition, I used UML class diagrams to provide in-depth detail for the packages, classes, variables, and method signatures needed to build the application.

    • VT100
      MIPS Project

      For my final project in my CPU Architecture and Assembly Language Programming class, I recreated a VT100 display in MIPS. To display the data, I used the Display MMIO (Memory-Mapped Input/Output) Simulator in MARS. The program uses commands to control tasks such as cursor control and background/foreground color changes. I also implemented a dictionary of common Java syntax, allowing for automatic text highlighting on the display.
